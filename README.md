# Smart Student Planner & ML-Based Scheduler

## Project Structure
- `frontend/`: React + Vite application.
- `backend/`: Node.js + Express API.
- `ml_service/`: Python + FastAPI service for ML and Scheduling.

## Setup

### Backend
```bash
cd backend
npm install
# Create .env file with DB credentials
npm start
```

### ML Service
```bash
cd ml_service
pip install -r requirements.txt
python main.py
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
